webmachine-tutorial
===================

Webmachine Tutorial for LambdaJam 2014.

## Setup
Before starting ensure you have erlang/otp installed, at least R16 and a unix
based OS.

 1. Clone project
 2. Run
```
./rebar get-deps
./rebar compile
```
 3. ./start.sh
